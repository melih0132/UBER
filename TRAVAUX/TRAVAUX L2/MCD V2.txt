Utilisateur:
  - id (clé primaire)
  - nom
  - prénom
  - email
  - mot de passe (chiffré)
  - type (particulier/entreprise)
  - photo
  - lieux de départ favoris
  - informations de carte bancaire (chiffrées)

Chauffeur:
  - id (clé primaire)
  - nom
  - prénom
  - statut (micro-entreprise, EI, société)
  - numéro de carte VTC
  - numéro de licence VTC
  - RIB
  - date de début d'activité
  - type de prestation (Green, XL, Pet, Confort, Berline, X)
  - note moyenne (calculée à partir des notes données par les clients)

Véhicule:
  - id (clé primaire)
  - immatriculation
  - modèle
  - couleur
  - type (berline, van, véhicule adapté)
  - catégorie (Green, XL, Pet, Confort, Berline, X)
  - chauffeur_id (clé étrangère référençant Chauffeur)

Course:
  - id (clé primaire)
  - date et heure de départ
  - lieu de départ
  - lieu d'arrivée
  - statut (demandée, en cours, terminée, annulée)
  - prix
  - pourboire
  - note chauffeur
  - commentaire note
  - client_id (clé étrangère référençant Utilisateur)
  - chauffeur_id (clé étrangère référençant Chauffeur)

Restaurant:
  - id (clé primaire)
  - nom
  - adresse
  - type de cuisine
  - temps estimé de livraison

Produit:
  - id (clé primaire)
  - nom
  - description
  - prix
  - catégorie
  - restaurant_id (clé étrangère référençant Restaurant)

Commande:
  - id (clé primaire)
  - date et heure de commande
  - type (livraison, à emporter)
  - statut (en attente, en préparation, livrée, annulée)
  - prix total
  - client_id (clé étrangère référençant Utilisateur)
  - restaurant_id (clé étrangère référençant Restaurant)

Livraison:
  - id (clé primaire)
  - date et heure de livraison
  - chauffeur_id (clé étrangère référençant Chauffeur)
  - commande_id (clé étrangère référençant Commande)

Vélo:
  - id (clé primaire)
  - numéro
  - statut (disponible, réservé)
  - localisation (adresse)

Réservation Vélo:
  - id (clé primaire)
  - date et heure de début
  - date et heure de fin
  - lieu de départ
  - lieu d'arrivée
  - client_id (clé étrangère référençant Utilisateur)
  - vélo_id (clé étrangère référençant Vélo)

Relations:
  - Un utilisateur peut effectuer plusieurs courses.
  - Un chauffeur peut effectuer plusieurs courses.
  - Un véhicule est associé à un seul chauffeur.
  - Une course est effectuée par un seul chauffeur et pour un seul client.
  - Un restaurant propose plusieurs produits.
  - Un utilisateur peut passer plusieurs commandes.
  - Une commande concerne un seul restaurant et un seul utilisateur.
  - Une commande peut être associée à une livraison.
  - Une livraison est effectuée par un seul chauffeur.
  - Un utilisateur peut effectuer plusieurs réservations de vélos.
  - Une réservation de vélo concerne un seul vélo et un seul utilisateur.

Attributs Dérivés:
  - Le prix total d'une commande peut être calculé à partir du prix des produits commandés.
  - La note moyenne d'un chauffeur peut être calculée à partir des notes données par les clients.

Contraintes:
  - Un chauffeur ne peut accepter qu'une seule course à la fois.
  - Un vélo ne peut être réservé que par un seul utilisateur à la fois.
